# Estilo Fotográfico SmartSenior

Diretrizes visuais para fotografia e imagens na marca SmartSenior, baseadas na análise do moodboard oficial.

---

## Filosofia Visual

A fotografia SmartSenior transmite **vitalidade, conexão e autonomia**. Nossas imagens mostram idosos ativos, engajados e felizes, sempre em contextos realistas e acolhedores.

> **Princípio central:** Mostrar a longevidade como uma fase de vida plena, não de limitação.

---

## Pessoas

### Características dos Modelos

| Aspecto | Diretriz |
|---------|----------|
| **Idade** | 60+ anos, aparência saudável e cuidada |
| **Expressão** | Sorrisos genuínos, olhares conectados |
| **Postura** | Ativa, engajada, não passiva |
| **Vestuário** | Casual elegante, cores vivas (laranjas, verdes, azuis) |
| **Diversidade** | Incluir diferentes etnias e tipos físicos |

### O Que Evitar

- Idosos em posição passiva ou dependente
- Expressões tristes ou preocupadas
- Roupas hospitalares ou muito formais
- Estereótipos de fragilidade

### Interações Ideais

- **Idoso + Familiar:** Momentos de conexão genuína (risos, abraços, conversas)
- **Idoso + Atividade:** Jardinagem, arte, culinária, exercícios leves
- **Idoso + Tecnologia:** Uso confiante de dispositivos

---

## Composição

### Enquadramento

| Tipo | Uso |
|------|-----|
| **Close-up** | Expressões faciais, detalhes de mãos em atividade |
| **Médio** | Interações entre pessoas, contexto da atividade |
| **Ambiente** | Estabelecer cenário, mostrar casa/jardim |

### Regras de Composição

1. **Regra dos terços:** Posicionar sujeitos nos pontos de interesse
2. **Espaço respirável:** Deixar margem para texto/overlays
3. **Profundidade:** Usar bokeh suave (f/2.8-4.0) para destacar sujeito
4. **Linhas guia:** Usar elementos do ambiente para guiar o olhar

### Aspect Ratios Recomendados

```
Feed Instagram:    1:1 (1080x1080) ou 4:5 (1080x1350)
Stories/Reels:     9:16 (1080x1920)
Website Hero:      16:9 (1920x1080) ou 21:9 (ultrawide)
Cards/Thumbnails:  3:2 (600x400) ou 4:3 (800x600)
```

## Iluminação

### Luz Natural (Preferida)

- **Janelas grandes:** Luz suave e difusa
- **Golden hour:** Tons quentes no início da manhã ou fim de tarde
- **Evitar:** Luz dura do meio-dia, sombras marcadas

### Ambiente

- Interiores bem iluminados
- Tons quentes predominantes
- Evitar ambientes escuros ou clínicos

### Temperatura de Cor

```
Alvo: 5500K - 6500K (luz do dia quente)
Evitar: Tons frios azulados
```

## Tratamento de Imagem

### Ajustes Recomendados

| Parâmetro | Ajuste |
|-----------|--------|
| **Exposição** | Levemente elevada (+0.3 a +0.5) |
| **Contraste** | Médio-baixo (suavizar sombras) |
| **Saturação** | Levemente elevada (+10 a +15%) |
| **Temperatura** | Quente (+5 a +10) |
| **Clareza** | Sutil (+5 a +10) |
| **Sombras** | Abertas (+20 a +30) |

## Overlays e Texto

### Padrões de Sobreposição

1. **Faixa Navy:** Barra inferior com `#003057` para texto/logo
2. **Texto Branco:** Sobre imagens escuras ou faixas coloridas
3. **Texto Laranja:** Para destaques e CTAs
4. **Gradiente:** Escurecimento gradual para legibilidade

### Posicionamento do Logo

- **Canto inferior direito:** Padrão para posts
- **Centro inferior:** Para materiais institucionais
- **Sempre sobre fundo contrastante**

### Hierarquia Tipográfica em Imagens

```
Headline:     Bold, 32-48px, Branco ou Laranja
Subheadline:  Medium, 18-24px, Branco
CTA Button:   Bold, 16-20px, Branco sobre Laranja
```

## Ambientes

### Cenários Ideais

| Ambiente | Elementos |
|----------|-----------|
| **Casa** | Cozinha aconchegante, sala iluminada, varanda |
| **Jardim** | Plantas, flores, luz natural filtrada |
| **Parque** | Natureza, bancos, caminhos arborizados |
| **Atividades** | Aulas de arte, grupos de exercício, encontros sociais |

### Elementos de Apoio

- Plantas e flores (vitalidade)
- Utensílios de cozinha (autonomia)
- Materiais de arte (criatividade)
- Dispositivos tecnológicos (modernidade)

### O Que Evitar

- Hospitais ou clínicas
- Ambientes escuros ou abandonados
- Cenários impessoais ou frios
- Mobiliário hospitalar

## Checklist de Qualidade

Antes de publicar qualquer imagem, verifique:

- [ ] Pessoa aparenta estar feliz e engajada?
- [ ] Iluminacao é quente e acolhedora?
- [ ] Ambiente transmite conforto e autonomia?
- [ ] Cores harmonizam com paleta SmartSenior?
- [ ] Logo está visível e bem posicionado?
- [ ] Texto tem contraste adequado (WCAG AA)?
- [ ] Imagem funciona nos diferentes aspect ratios?

## Banco de Imagens

### Fontes Recomendadas

Para imagens de estoque que sigam estas diretrizes:

1. **Getty Images** - Buscar "active seniors", "happy elderly"
2. **Shutterstock** - Filtrar por "lifestyle", "warm tones"
3. **Adobe Stock** - Coleções de envelhecimento ativo
4. **Unsplash** - Gratuito, buscar "elderly gardening", "grandparents"

### Palavras-chave de Busca

```
English: active seniors, happy elderly, multigenerational family,
         senior lifestyle, elderly gardening, grandparents cooking

Portugues: idosos ativos, terceira idade feliz, familia multigeracional,
           avos cozinhando, jardinagem seniores
```
